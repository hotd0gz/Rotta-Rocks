### View Firewall Rules

In UNIX systems _**sudo**_ permissions is required to use _**iptables**_. We may be able to circumvent this by viewing the rules file directly.&#x20;

```shell-session
~$ cat /etc/iptables/rules.v4
# Generated by xtables-save v1.8.2 on Thu Aug 18 12:53:22 2022
*filter
:INPUT ACCEPT [0:0]
:FORWARD ACCEPT [0:0]
:OUTPUT ACCEPT [0:0]
-A INPUT -p tcp -m tcp --dport 1999 -j ACCEPT
COMMIT
# Completed on Thu Aug 18 12:53:22 2022
```

# Edit this file to introduce tasks to be run by cron.
